# Vue全家桶学习路线

前言

大概介绍一下Vue全家桶学习的方向。

# Vue-CLI 脚手架

Vue CLI 是官方提供的一个脚手架，用于快速生成一个 vue 的项目模板，类似于 Spring Initializr。

# Vue-CLI 3.x版本后变动

Vue3.x创建项目

vue create hello-world
1

Vue CLI> = 3和旧版使用了相同的vue命令，所以Vue CLI 2（vue-cli）被覆盖了。如果您仍然需要使用旧版本的vue init功能，您可以安装一个插入工具

npm install -g @vue/cli-init
# `vue init` 的运行效果将会跟 `vue-cli@2.x` 相同
vue init webpack my-project

# Vue 基础

Vue 基础语法类似于Thymeleaf，都是直接在html元素中操作[v-if,v-for]，JSTL则是标签。
查看文档

这里推荐官网推荐的DCloud视频教程 查看视频，通过引入CDN的方式差不多一个小时的样子就能把这些过一遍，熟悉一下vue的语法。

# Axios [Vue-resource] 通信

Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。
Vue-resource 官方团队停止维护，Vue.js 2.0 版本推荐使用 axios 来完成 ajax 请求。
查看文档

# Vue Router 路由

Vue Router 是 Vue.js 官方的路由管理器。它和 Vue.js 的核心深度集成，让构建单页面应用变得易如反掌。
查看文档

# Vuex 状态管理

Vuex 是一个专为 Vue.js 应用程序开发的状态管理模式。它采用集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以一种可预测的方式发生变化。
查看文档

# Vue-Devtools 浏览器插件

浏览器devtools扩展，用于调试Vue.js应用程序。
查看文档
